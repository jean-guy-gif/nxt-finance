# Design direction - NXT Finance

## Objective
Build NXT Finance with a premium admin dashboard feel inspired by the following Figma reference:
https://www.figma.com/design/b4dMPT7NfDYfb1fCnQYRLt/NEATLAB---Super-Admin-Dashboard-UI-Design-Kit--Community-?node-id=23499-134071&m=dev&t=04x1DP6Sh6vRKPvL-1

This Figma file is an inspiration source for:
- layout rhythm
- dashboard density
- card hierarchy
- spacing
- information architecture
- visual polish
- component composition
- admin/SaaS feel

It must NOT be copied literally.

## Brand priority
The final UI must match the brand identity and product feeling of NXT Profiling:
https://nxt-profiling.fr/

NXT Profiling is the brand and visual reference to respect.
The Figma kit is only a structural and UI inspiration reference.

## Design translation rules
Claude must:
- keep the overall premium SaaS dashboard quality of the Figma reference
- adapt all colors, visual identity, typography feeling, and product personality to the NXT Profiling brand
- preserve a clean, modern, executive, reassuring interface
- avoid a generic template feel
- create a product that feels like a native extension of NXT Profiling, not a reused dashboard kit

## Use the Figma reference for
- page layout inspiration
- section composition
- sidebar/topbar balance
- dashboard card organization
- table and filter layout
- spacing between components
- visual hierarchy of KPI blocks
- modal and panel structure
- general premium dashboard standards

## Do not reuse directly
- exact colors
- exact icons if they conflict with brand tone
- exact typography if it conflicts with brand tone
- exact illustrations
- exact gradients
- exact shadows if they feel off-brand
- any element that makes the UI look like a template clone

## Brand adaptation requirements
The final design must feel:
- more aligned with NXT Profiling
- more business-oriented for agency managers
- simpler and more operational than a generic super-admin
- clearer for finance and pre-accounting workflows
- elegant, modern, and trustworthy

## Product-specific UI principles
NXT Finance is not a generic analytics dashboard.
It is a finance and pre-accounting cockpit for real estate agency managers.

The interface must emphasize:
- clarity
- speed of understanding
- trust
- operational readability
- actionability
- low cognitive load

## Visual priorities
Prioritize:
1. readability of financial information
2. alert visibility
3. quick actions
4. period navigation
5. expense / receipt / VAT workflow clarity
6. accountant collaboration visibility

## Components that need special care
- financial KPI cards
- alert banners
- period status chips
- expense tables
- receipt upload areas
- VAT summary blocks
- accountant request panels
- multi-device responsive layouts

## Responsive behavior
The design must work across:
- desktop
- tablet
- mobile

Mobile priority:
- upload ticket
- upload invoice
- quick validation
- alert consultation

Desktop priority:
- dashboard overview
- table workflows
- period review
- accountant collaboration

## Expected outcome
The final result must look like:
- a premium NXT product
- inspired by a strong SaaS dashboard system
- visually coherent with NXT Profiling
- not like a direct clone of the Figma template
